## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn

### Installation

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm run dev
```

3. Open your browser and navigate to `http://localhost:3000`

### Build for Production

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```

## Customization

### Colors
Edit the CSS variables in `src/index.css`:
```css
:root {
  --primary-color: #FFD700;
  --secondary-color: #1a1a2e;
  --accent-color: #16213e;
  /* ... more colors */
}
```

### Content
Update the content in each component file to match your requirements.

### Animations
Modify Framer Motion animations by adjusting the `motion` component props in each component.

## Animation Features

- **Page Load Animations**: Smooth fade-in and slide effects
- **Scroll Animations**: Elements animate as you scroll

- **Background Animations**: Subtle moving gradients

- **Floating Elements**: Continuous floating animations
- **Scale Transformations**: Grow/shrink effects
- **Gradient Movements**: Animated color transitions
